# quickshell-DynamicIsland
基于Archirithm的项目二次修改而来⬇️
https://github.com/Archirithm/dotfiles

⚠️我只用到了quicksheel基本功能，这里只介绍我用到的功能，具体的使用可以参考原作者。

==============================================

主要就是增加了蓝牙连接（需要安装bluetui！！！）

移除了cava音量频条（可以加回来,在/quickshell/Modules/Bar/Bar.qml第60行移除注释就可以）

系统状态改为轮播切换

封面可以自动根据歌名和歌手自动从网络爬取

使用方法：直接替换～/.config/quickshell文件夹即可

===============================================

目录注释：

/.config/quickshell/Modules/Bar/ 存放左中右小组件，增删改左右小组件在这

/.config/quickshell/Modules/DynamicIsland/ 存放灵动岛小组件，改灵动岛在这

/.config/quickshell/scripts/ 存放调用的脚本

不保证每台电脑界面都适配，有问题可以自行研究修改
